# [GraphicsLab] - CG Algorithm Visualizer

A browser-based visualizer for classic Computer Graphics scan-conversion algorithms. Enter coordinates, pick an algorithm, and watch it plot points step-by-step on an interactive grid canvas.

---

## Features

- **Interactive canvas** with a centered coordinate system, zoomable grid, and axis labels
- **Hover tooltips** showing the exact `(x, y)` of every plotted point
- **Animated drawing** - watch the algorithm place points one by one
- **Point log** listing every computed coordinate in order
- **Export** the canvas output as a PNG
- **Clean, dark retro-futuristic UI** - no frameworks, no dependencies


## Getting Started

No build step required. Just open `index.html` in any modern browser.
